# 🎓 Banco de Dados Acadêmico

## 📌 Descrição do Projeto
Este repositório contém a estrutura de um banco de dados acadêmico desenvolvido para fins acadêmicos. O objetivo é organizar e gerenciar informações relacionadas a alunos, professores, cursos e notas.

O banco de dados foi projetado para permitir a administração eficiente das entidades acadêmicas e seus relacionamentos, garantindo integridade e escalabilidade.

## 📁 Estrutura do Repositório
- **📜 queries.sql** → Contém os comandos SQL para criar as tabelas e os relacionamentos do banco de dados acadêmico.

## 🚀 Tecnologias Utilizadas
- **Banco de Dados:** MySQL
- **Linguagem SQL:** Structured Query Language (SQL)
- **Ferramentas:** Workbench / HeidiSQL / DBeaver (ou outra ferramenta de administração de banco de dados)
